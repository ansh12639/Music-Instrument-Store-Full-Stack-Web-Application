🎵 Music Instrument Store – Full-Stack Web Application

A full-stack e-commerce platform for buying and managing music instruments, built using PHP, MySQL, HTML, CSS, and JavaScript. The system integrates frontend UI with backend logic and a relational database to simulate a production-level online retail workflow.

📊 Project Highlights

🎸 20+ dynamic instrument listings stored in MySQL
👤 User registration & login authentication system
🛒 Real-time cart management with automated total calculation
📦 Order placement & database-driven transaction handling
🛠️ Admin panel with CRUD operations for products
🔐 Server-side validation and secure session management
💬 System Capabilities


Database-integrated product catalog
Category-based browsing (Guitar, Piano, Drums, etc.)
Responsive UI for seamless user interaction
Structured relational schema with normalized tables
Local deployment via XAMPP (Apache + MySQL)
